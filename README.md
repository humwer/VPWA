# VPWA
## Vulnerable Portal Web Application

Проект экзамена по базовым навыкам проведения тестирования на проникновение в веб-приложение.

<details>
<summary>В экзамене присутствуют следующие уязвимости:</summary>
  
>1. **Re-registration users**
>2. **Bruteforce**
>3. **XXE**
>4. **SQLi**
>5. **SSTI**
>6. **XSS & Cookie without HttpOnly**
>7. **Path Traversal**
>8. **SSRF**
>9. **Insecure token refresh**

</details>

И несколько не упомянутых :)

## Installation
> 1. git clone git@github.com:humwer/VPWA.git
> 2. docker-compose build
> 3. docker-compose up
